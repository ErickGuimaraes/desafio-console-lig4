# Console Lig-4

![Console Lig-4](/ConsoleLig4.png?raw=true "Console Lig-4")

Developed in dotnet 5.

Build and run using dotnet CLI:
- dotnet restore
- dotnet build -c Release -o _build
- dotnet _build/ConsoleLig4.dll

## Requirements
- .Net SDK 5 installed
